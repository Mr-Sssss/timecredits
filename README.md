# Timecredits
## Eine bessere Website für das Timecredits System für das Käthe-Kollwitz-Gymnasium. 

> Wir sind drei Schüler auf dem Käthe-Kollwitz-Gymnasium und besuchen die zehnte Jahrgangsstuffe. Wir haben uns die Aufgabe gesetzt die Website für das Timecreditsystem dieser Schule zu verbessern.
> Momentan ist es recht kompliziert. Auf unserer Schule muss man bestimmte Ganztags- und Berufs- und Studiumorientierungsangebote machen, die jeweils bestimmte Punkte bringen. Am ende des Schuljahres muss man 100 Punkte erreichen.

Momentan muss man auf einer Website gehen wo man die Angebote sehen kann wann und wie die stattfinden.
Dann geht man auf eine seperate Website um das Angebot zu buchen. Diese Website ist aber nicht optimiert, da diese eine Vorlag verwendet was eigentlich für einen fucking Online-Shop gedacht ist. Dann "kauft" man das Angebot muss den Kaufbeweis screenshoten auf einer dritten random Website hochladen mit einem nicht definierten Beweis, dass man dieses Angebot wirklich gemacht hat. Man weiß nicht mal wie viele Punkte man hat, welceh Beweise gültig sind und auch sind die nachguck-Website und die Buchungs-website nicht aktuell, da es Angebote auf der Nachguck-Website gibt, die man machen will, diese aber nicht auf der Buchungs-Website sind, also es nicht möglich ist dieses Angebot zu machen, da man kein Buchungsbeweis einreichen kann.
