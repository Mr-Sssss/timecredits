# Timecredits
## Eine bessere Website für das Timecredits System für das Käthe-Kollwitz-Gymnasium. 
  
  
Wir sind drei Schüler auf dem Käthe-Kollwitz-Gymnasium und besuchen die zehnte Jahrgangsstuffe. Wir haben uns die Aufgabe gesetzt die Website für das Timecreditsystem dieser Schule zu verbessern.
Momentan ist es recht kompliziert. Auf unserer Schule muss man bestimmte Ganztags- und Berufs- und Studiumorientierungsangebote machen, die jeweils bestimmte Punkte bringen. Am ende des Schuljahres muss man 100 Punkte erreichen, sonst kann eine Schulvesäumnissanzeige drohen.
  
 
Momentan muss man auf einer Website gehen wo man die Angebote sehen kann wann, wie und wo die stattfinden.
Dann geht man auf eine seperate Website um das Angebot zu buchen. Diese Website ist aber nicht optimiert, da diese eine Vorlag verwendet was eigentlich FÜR EINEN FUCKING ONLINE-SHOP gedacht ist. Dann "kauft" man das Angebot, muss den Kaufbeweis screenshoten, diese dann aud einer dritten random Website hochladen. Zusätzlich muss man einen "Beweiß" hochladen, um zu zeigen, dass dieses Angebot auch wirklich gemacht wurde. Das Problem dabei ist aber, man weiß nicht welche bewiese gültig sind und auch welche bewiese überhaupt von einem erwartet werden. Man weiß nicht mal wie viele Punkte man hat, welche Beweise überhaupt schon überprüft worden. Auch sind die "nachguck-Website" und die "Buchungs-website" nicht aktuell. Es Angebote auf der Nachguck-Website, die man vieleicht machen will, diese aber nicht auf der Buchungs-Website sind. Also ist es nicht möglich ist dieses Angebot zu machen, da man kein Buchungsbeweis einreichen kann.
  
 
Wie ihr seht ist dieses System ein völliges durcheinander und genau das wollen wir verbessern. Wir wollen eine Website erschaffen wo man direkt alle Angebote sehen, buchen und auch direkt den definierten Beweis einreichen kann. Es gibt dann eine Tabelle wo man alle Angebote sehen, Buchen und Beweisen kann. Den Status des Angebotes also z.B. "gebucht", "Beweis wird geprüft", "Beweis bestätigt", "Beweis abgelehnt", usw. wird dann auch angezeigt. Man kann auch sehen wie viele Punkte man schon hat, auch aufgeteilt in "Punkte nur mit bestätigten Beweis" oder "Punkte die eingereicht wurden".

  
Damit schaffen ein besserse und Organisertes System mit dem jeder mitkommen kann und es auch versteht.
